Gerenciador de Tarefas com Login

Sistema web de gerenciamento de tarefas no estilo Kanban, com autenticação de usuários e persistência de dados no navegador.

Funcionalidades

*  Cadastro e login de usuários
*  Criação de tarefas
*  Organização em colunas (A Fazer, Em Progresso, Concluído)
*  Movimentação de tarefas entre colunas (drag and drop)
*  Exclusão de tarefas
*  Armazenamento local com LocalStorage

Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* LocalStorage
